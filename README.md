# 🚀 SPACE GAME - Evolution Project\n\n> **Real-time space strategy evolving from Galcon simplicity to 4X depth**\n\n[![Status](https://img.shields.io/badge/Status-Action%2002%20Complete-success)](#)\n[![Balance](https://img.shields.io/badge/Balance-Experimenting-yellow)](#)\n[![Version](https://img.shields.io/badge/Version-1.4.5-blue)](#)\n\n## 🎮 **What is Space Game?**\n\nAn **evolutionary strategy game** that starts with familiar Galcon mechanics and gradually adds 4X strategy depth through successive \"Actions\" (development phases).\n\n### **Core Philosophy: \"Opción A Galcon\"**\n- 🆓 **Ships regenerate FREE** on owned planets\n- ⚡ **Moving fleets costs energy** (fuel system)\n- 🏗️ **Buildings provide strategic advantages**\n- 🌍 **Distance matters** for tactical decisions\n\n---\n\n## 🎯 **Current State - Action 02 Complete**\n\n### ✅ **Implemented Features**\n- **Complete building system** with 3 building types\n- **AI opponent** with intelligent building decisions\n- **Resource economy** (Metal for construction, Energy for movement)\n- **Visual feedback** with building indicators and animations\n- **Balanced gameplay** averaging 5-8 minute games\n\n### 🔄 **Active Development**\n- **Energy as Fuel System** - Transform energy into tactical movement fuel\n- **Distance-based costs** - Geography affects strategy\n- **Resource role separation** - Metal = construction, Energy = movement\n\n---\n\n## 🚀 **Quick Start**\n\n### **Play Now**\n1. Open `index.html` in any modern browser\n2. Select Classic mode\n3. Left-click planets to select, right-click to send fleets\n4. Right-click your planets to build structures\n\n### **Controls**\n- **Left Click**: Select planet\n- **Right Click**: Send fleet / Open building menu\n- **Keyboard**: Assigned keys for quick planet selection\n\n---\n\n## 📁 **Project Structure**\n\n```\nspaceGame/\n├── 📁 src/           # Organized source code\n│   ├── core/         # Game engine core\n│   ├── systems/      # Buildings, resources, AI\n│   ├── ui/           # User interface\n│   └── config/       # Game configuration\n├── 📁 docs/          # Documentation\n│   ├── action-01/    # Action 01 archive\n│   ├── action-02/    # Action 02 archive\n│   ├── balance/      # Balance experiments\n│   └── planning/     # Future roadmap\n├── 📁 css/           # Stylesheets\n├── 📁 js/            # Current JS code\n└── index.html        # Game entry point\n```\n\n---\n\n## 🛠️ **Development**\n\n### **Current Branch**\n`action-02-balance-experiments` - Energy fuel system implementation\n\n### **Key Files**\n- `js/config.js` - Main game configuration\n- `js/balanceConfig.js` - Balance parameters\n- `js/gameEngine.js` - Core game loop\n- `js/buildings.js` - Building definitions\n\n### **Testing Balance Changes**\n1.
